## Descripción

Este repositorio contiene la resolución del **Trabajo Práctico N°2** de la materia **Aprendizaje Automático II (AA2)** de la **Tecnicatura Universitaria en Inteligencia Artificial**.

El trabajo se compone de dos problemas principales:

### **Ejercicio 1: Audio MNIST**
Construcción de modelos de clasificación de dígitos hablados (0-9) utilizando redes neuronales. Se implementan dos arquitecturas:
- Modelo convolucional sobre espectrogramas
- Modelo recurrente sobre espectrogramas

### **Ejercicio 2: Flappy Bird**
Entrenamiento de agentes para jugar Flappy Bird utilizando técnicas de aprendizaje por refuerzo:
- Agente con Q-Learning tradicional
- Agente con Deep Q-Learning

## Instrucciones Generales

### Clonar el repositorio

Para clonar el repositorio ejecuta el siguiente comando en la terminal:

```bash
git clone --depth 1 https://github.com/FabricioPorcelli/AA2-TP2
cd AA2-TUIA-TP2
```

### Requisitos Previos

- Python 3.12 o superior
- pip (gestor de paquetes de Python)
